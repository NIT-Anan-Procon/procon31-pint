# procon31 Pint

## 開発環境

* Ubuntu Server 20.04

* Node.js 12.84.4

	* npm 6.14.6

	* yarn 1.22.5
	
	* React 16.13.1
	
	* axios 0.19.2
	
	* react-youtube 7.11.3
	
## 実行環境

* Google Chrome 最新版

## このリポジトリについて

* Reactのプロジェクトです。

## 実行するには

### `npm install`

依存ファイル、コンポーネントを同期・置き換えます。

### `yarn start`

開発モードで動きます。

### `yarn build`

Reactのプロジェクトファイルを出力します。
